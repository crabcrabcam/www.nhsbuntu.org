--- title: Technical Details ---

Based on 16.04 Ubuntu Linux and the 4.4 kernel for long term support

Using the Gnome desktop for ease of use and a clean design


Contains the following packages:

- Evolution-EWS Email Client

- Mozilla Firefox ESR Web Browser

- LibreOffice Suite

- WINE Compatability Layer



Extra details

Evolution-EWS Email Client is an email client that is set up to work with NHSMail2

Google Chrome install scripts are provided if you wish to use them

Mozilla Firefox is provided fully set up as a browser

The LibreOffice suite is a toolset like Microsoft Office for working with text documents, slideshows or spreadsheets.

WINE Compatability Layer is a tool that can run Windows programs on Linux


Hardware Specifications

NHSbuntu runs on quite low end hardware.
